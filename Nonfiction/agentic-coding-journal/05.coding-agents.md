`pi install npm:pi-mcp-adapter`

**Security**
Restrict access to your file system in `~/.pi/agent/mcp.json`

```
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/russellwarner/Development"
      ]
    }
  }
}
```
