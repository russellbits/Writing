# An Agentic Content Management System

With little fanfare, I mean for this system to replace Wordpress. Period. I began using Wordpress when it was a few years old. It gave some significant benefits back then. Now, it is a major target of attack because of the number of large web sites that use it. Its database has become an architectural nightmare. It isn't very fast and despite an enormous number of plugins and design frameworks, it is actually very difficult for a **novice** to use. It requires a level of technical expertise that drives users from their own web sites on to platforms, where it is easy for users to post content.

+ This system is also anti-SPA.
+ It is anti-lists-n-boxes in that it has a natural language interface.
